# Employee Portal React

React frontend scaffold for the existing employee portal backend.

## Setup

1. Open this folder:
   `employee-portal-react`
2. Install dependencies:
   `npm install`
3. Start the dev server:
   `npm run dev`

## API base URL

By default the app uses:

- production: `https://employee-portal-backend-zqbo.onrender.com`
- local: `http://localhost:3000`

You can override this with:

`VITE_API_BASE_URL=http://localhost:3000`

Create a `.env` file in this folder if needed.
